# <a name="use-your-browser-to-check-spelling"></a>Utilizzare il browser per il controllo ortografico

Non esiste alcuna opzione di controllo ortografico in Outlook.com. Per controllare l'ortografia, sarà necessario utilizzare il browser web. Il controllo ortografico è disponibile in Microsoft Edge, Internet Explorer 10 e versioni successive e le versioni correnti di Firefox, Chrome e Safari. Controllare le opzioni per il browser web per ulteriori informazioni su come eseguire il controllo ortografico.
