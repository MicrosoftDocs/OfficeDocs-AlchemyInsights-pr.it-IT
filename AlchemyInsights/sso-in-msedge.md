# <a name="single-sign-on-sso-in-microsoft-edge"></a>Single #A0 (SSO) in Microsoft Edge

Microsoft Edge supporta i seguenti tipi di single sign-on (SSO):
- SSO basato su PRT
- SSO senza problemi
- SSO basato su WIA
