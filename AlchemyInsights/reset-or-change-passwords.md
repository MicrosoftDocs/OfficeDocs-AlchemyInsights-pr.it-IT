# <a name="reset-or-change-passwords"></a>ReImpostare o modificare le password

Un utente può modificare la propria password dall' [account, la privacy &amp; della sicurezza](https://portal.office.com/account/#security).
  
Se l'utente ha dimenticato la password, un amministratore può reimpostare la password accedendo agli utenti [ \> attivi](https://portal.office.com/adminportal/home#/users), selezionare gli utenti e fare clic su **Reimposta password**. 
  
[Fare clic qui per ulteriori informazioni](https://support.office.com/article/admins-reset-office-365-business-passwords-7a5d073b-7fae-4aa5-8f96-9ecd041aba9c)o per consentire agli utenti di reimpostare le proprie password dimenticate.
