# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Recupero di un account disabilitato errore in Outlook sul web?

Se viene visualizzato l'errore **è stato disabilitato l'account** con `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` quando si tenta di aprire Outlook sul web (precedentemente noto come Outlook Web App), l'amministratore dell'organizzazione sia stato disabilitato l'accesso a Outlook sul web. L'amministratore può risolvere l'errore seguendo i passaggi nell'argomento [abilitare o disabilitare Outlook Web App per una cassetta postale](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx).
