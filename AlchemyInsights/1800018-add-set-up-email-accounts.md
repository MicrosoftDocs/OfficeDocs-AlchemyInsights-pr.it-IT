# <a name="problems-setting-up-an-exchange-email-account-in-outlook"></a>Problemi durante l'impostazione di un account di posta elettronica di Exchange in Outlook

Se si verificano problemi durante l'impostazione di un account di Exchange in Outlook, è possibile eseguire lo [strumento di installazione di Outlook account](https://aka.ms/SaRA-OutlookSetupProfile) del computer interessato. Eseguire controlli diversi e offrire soluzioni possibili per il problema di configurazione di account. 
  
Se si riscontrano ancora problemi, vedere [che Outlook non è impostato un nuovo profilo utilizzando l'individuazione automatica di Exchange per una cassetta postale di Exchange Online in Office 365](https://support.microsoft.com/en-us/help/2404385/outlook-can-t-set-up-a-new-profile-by-using-exchange-autodiscover-for).
