# <a name="use-microsoft-edge-to-remotely-debug-an-android-device-from-a-development-computer"></a>Utilizzare Microsoft Edge per eseguire il debug remoto di un dispositivo Android da un computer di sviluppo

Per eseguire il debug remoto di contenuto attivo su un dispositivo Android dal computer in uso:

1. Configurare il dispositivo Android per il debug remoto e individuarlo dal computer di sviluppo.
2. Dal computer di sviluppo, ispezionare e eseguire il debug di contenuto Live sul dispositivo Android.
3. Screencast sul contenuto live del dispositivo Android su un'istanza di DevTools nel computer di sviluppo.
