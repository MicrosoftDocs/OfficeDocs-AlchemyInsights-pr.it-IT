# <a name="map-a-sharepoint-library-to-a-network-drive"></a>Eseguire il mapping di una raccolta SharePoint a un'unità di rete

Per utilizzare raccolte siti in Esplora File, è consigliabile utilizzare il client di sincronizzazione OneDrive anziché eseguire il mapping delle librerie per le unità di rete. Mapping dell'unità è solo una soluzione temporanea. Utilizzando il client di sincronizzazione rende i file disponibili per gli utenti anche quando è in linea. Il client di sincronizzazione anche garantisce prestazioni migliori, consente l'accesso ai file senza la necessità di scaricarli utilizzando [i file di richiesta](https://support.office.com/en-us/article/Learn-about-OneDrive-Files-On-Demand-0E6860D3-D9F3-4971-B321-7092438FB38E) e non richiede il programma di installazione utilizza Internet Explorer. 
  
- [Sincronizzare i file con il nuovo client di sincronizzazione di OneDrive in Windows](https://go.microsoft.com/fwlink/?linkid=866427).
    
- [Come gestire i file delle raccolte siti in Esplora file](https://go.microsoft.com/fwlink/?linkid=866291)
    
- [Configurare e risolvere le unità di rete](https://support.microsoft.com/kb/2616712)
