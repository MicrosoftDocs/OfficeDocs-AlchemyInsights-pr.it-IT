# <a name="report-on-file-and-folder-sharing-in-sharepoint-sites"></a>Relazione sulla condivisione di file e cartelle nei siti di SharePoint

È possibile creare un file CSV in cui vengono illustrati tutti i file, gli utenti, le autorizzazioni e i collegamenti univoci in un determinato sito di SharePoint o in una raccolta di OneDrive. Questo può essere utile per comprendere il modo in cui viene usata la condivisione e se i file o le cartelle vengono condivisi con utenti guest.

Per ulteriori informazioni, [segnalare la condivisione di file e cartelle in un sito di SharePoint](https://docs.microsoft.com/sharepoint/sharing-reports).

> [!NOTE]
> Questa funzionalità è attualmente distribuita ai tenant del primo rilascio e verrà gradualmente implementata in tutti i tenant di produzione.
