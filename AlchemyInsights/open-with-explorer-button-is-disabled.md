# <a name="the-open-with-explorer-button-is-disabled"></a>Apri con il pulsante Explorer sono disattivato

- Selezionare il percorso di SharePoint o OneDrive in Internet Explorer 10 o 11 di Internet Explorer. Il pulsante non è compatibile con altri web browser.
    
> [!NOTE]
> Apertura di una raccolta in Esplora File è utile se si desidera copiare o spostare più file e cartelle una volta, ma se si desidera utilizzare regolarmente nella raccolta, è consigliabile la sincronizzazione. Per ulteriore assistenza e di altri problemi aprire Esplora File, vedere [Open nell'elenco cartelle](https://go.microsoft.com/fwlink/?linkid=871665). Per informazioni sull'impostazione di sincronizzazione, vedere [i file di sincronizzazione di SharePoint con il nuovo client di sincronizzazione OneDrive](https://go.microsoft.com/fwlink/?linkid=871666).
