# <a name="removing-the-last-on-premises-exchange-server"></a>Rimozione dell'ultimo server Exchange locale

Prima di rimuovere l'ultimo server di Exchange dall'organizzazione locale, è necessario assicurarsi di aver compreso le implicazioni di tale operazione per decidere se è necessario l'ultimo server per altri scopi. Per ulteriori informazioni, vedere [How and when to decommission your on-premises Exchange Servers in a Hybrid Deployment](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx).
