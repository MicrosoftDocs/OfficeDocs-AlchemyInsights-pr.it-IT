# <a name="change-your-password"></a>Modificare la password

La password di Outlook.com è identica alla password dell'account Microsoft.
  
Per modificare la password, passare a [https://account.microsoft.com/security](https://go.microsoft.com/fwlink/p/?linkid=842325&amp;clcid=0x409) e selezionare **Cambia password**. 
  
Se si è dimenticato la password, passare alla [pagina di reimpostazione della password dell'account Microsoft](https://go.microsoft.com/fwlink/p/?linkid=841909).
  
Per ulteriori informazioni, vedere [Change your password in Outlook.com](https://go.microsoft.com/fwlink/?linkid=873109).
