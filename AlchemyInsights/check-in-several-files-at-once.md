# <a name="check-in-several-files-at-once"></a>Archiviare più file contemporaneamente

Per controllare più file nelle cartelle diverse, selezionare **Mostra tutti gli elementi senza cartelle** in **cartelle** nella pagina **Modifica visualizzazione** . 
  
 **Nota**: SharePoint Online e Office 365 attualmente non consentono più file per il controllo con modalità moderna. Utilizzare esperienza classica facendo clic su **SharePoint classica** in basso a sinistra della barra di avvio veloce. 
  
Vedere più domande frequenti relative al controllo-in/check-out, vedere [domande frequenti relative a check-out, archiviazione e le versioni](https://go.microsoft.com/fwlink/?linkid=2018786).
