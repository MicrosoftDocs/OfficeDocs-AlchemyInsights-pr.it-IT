# <a name="setting-up-forwarding-to-multiple-email-addresses"></a>Impostazione di inoltro a più indirizzi di posta elettronica

Per prima cosa, passare a [gestire i gruppi](https://portal.office.com/adminportal/home#/groups)
  
1. Creare una *lista di distribuzione* 
    
2. Aggiungere più utenti
    
3. Quindi configurare l'inoltro delle cassette postali a tale gruppo
