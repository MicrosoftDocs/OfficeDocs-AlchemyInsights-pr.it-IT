# <a name="submit-dynamics-365-support-requests"></a>Inviare le richieste di assistenza Dynamics 365

Se si è un cliente di **Engagement per i clienti Microsoft Dynamics** inviare i ticket tramite la nuova [piattaforma Power Admin Center](https://admin.powerplatform.microsoft.com/?ref=officemodern).
  
Il nuovo portale offre un percorso più diretto di supporto assistito ottimizzata per engagement per i clienti Dynamics e altri prodotti della piattaforma Power.
