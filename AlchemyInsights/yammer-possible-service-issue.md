# <a name="yammer---possible-service-issue"></a>Yammer - possibile problema del servizio

Potrebbe verificarsi un problema temporaneo in cui gli utenti potrebbero verificarsi errori durante l'accesso a Yammer. Controllare il [dashboard di integrità del](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) servizio per verificare se ciò influisce sulla rete.
