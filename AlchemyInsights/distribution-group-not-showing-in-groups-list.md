# <a name="distribution-group-not-showing-in-groups-list"></a>Gruppo di distribuzione non viene visualizzata nell'elenco gruppi

È possibile richiedere fino a 60 minuti prima che venga visualizzato nuova lista di distribuzione nell'interfaccia di amministrazione.
  
Se si desidera aggiungere membri o apportare altre modifiche direttamente, utilizzare la funzionalità gruppi nell' [Interfaccia di amministrazione di Exchange](https://outlook.office365.com/ecp/?rfr=Admin_o365&amp;exsvurl=1).
