# <a name="set-up-microsoft-information-governance"></a>Configurare la governance delle informazioni di Microsoft

La [Guida all'installazione di Microsoft Information Governance](https://admin.microsoft.com/AdminPortal/Home#/modernonboarding/migsetupguide) fornisce le informazioni necessarie per configurare e gestire la strategia di governance dell'organizzazione, per assicurarsi che i dati siano classificati e gestiti in base alle linee guida specifiche del ciclo di vita impostate.

Con questa guida, vengono fornite informazioni su come creare, applicare automaticamente o pubblicare etichette, criteri per le etichette e criteri di conservazione applicati ai record di conformità e contenuto riutilizzabili dell'organizzazione. È inoltre possibile ottenere informazioni sull'importazione di file CSV con un piano di file per gli scenari di massa o per applicarli manualmente ai singoli documenti.
