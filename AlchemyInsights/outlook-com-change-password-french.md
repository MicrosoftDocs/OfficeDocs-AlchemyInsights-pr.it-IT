# <a name="changer-votre-mot-de-passe-dans-outlookcom"></a>Changer votre mot de passe dans Outlook.com

Votre mot de passe Outlook.com est le même que votre mot de passe compte comp Microsoft.

1. Accédez à [https://account.microsoft.com/security](https://go.microsoft.com/fwlink/p/?linkid=842325&amp;clcid=0x409) et Sélectionnez **modifier le mot de passe**.
2. Par mesure de sécurité, vous pouvez être invité à confirmer votre identité à l'aide d'un code de sécurité.
3. Entrez votre mot de passe Actuel, votre nouveau mot de passe, puis Sélectionnez ****.

Si vous avez oublié votre mot de passe, accédez à la [Page de réinitialisation de mot de passe de compte Microsoft](https://go.microsoft.com/fwlink/p/?linkid=841909), puis suivez les invita.

Si vous pensez que votre Outlook.com a été piraté, voir [Mon compte Outlook.com a](https://support.office.com/fr-fr/article/mon-compte-outlook-com-a-été-piraté-35993ac5-ac2f-494e-aacb-5232dda453d8)été piraté.
