# <a name="your-users-mailbox-is-being-created"></a>Creazione della cassetta postale dell'utente

È possibile richiedere un'ora prima che una nuova cassetta postale è pronta per l'utilizzo.
  
Cassetta postale dell'utente è stato "preparazione" per più di un'ora, è possibile provare eliminazione e ricreazione dell'utente o utilizzare l'opzione "possiamo chiamare l'utente" per parlare con un responsabile del supporto tecnico.
