# <a name="create-users-in-hybrid-deployments"></a>Creare utenti nelle distribuzioni ibride

Quando l'organizzazione è una distribuzione ibrida tra un ambiente locale e Office 365, è necessario creare tutti gli oggetti utente in Active Directory locale. È possibile creare gli oggetti utente nell'unità organizzativa appropriata in modo che siano sincronizzati nell'organizzazione di Office 365.
