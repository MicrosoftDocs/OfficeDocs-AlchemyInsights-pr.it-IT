# <a name="learn-about--the-mapping-between-microsoft-edge-legacy-policies-and-microsoft-edge-policies"></a>Informazioni sul mapping tra criteri della Versione legacy di Microsoft Edge e criteri di Microsoft Edge

I Criteri di gruppo della Versione legacy di Microsoft Edge (versioni 45 e precedenti) e le impostazioni di Gestione dispositivi mobili (MDM) vengono associati tramite mapping ai criteri pertinenti di Microsoft Edge, versione 80. Per altre informazioni, vedere [Mapping dei criteri della Versione legacy di Microsoft Edge a Microsoft Edge](https://docs.microsoft.com/deployedge/microsoft-edge-policy-map-legacy-to-newedge).
