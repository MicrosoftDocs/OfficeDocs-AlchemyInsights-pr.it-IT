# <a name="microsoft-teams---guest-access"></a>Team di Microsoft - accesso Guest

Se si desidera abilitare o configurare il ruolo accesso Guest in Microsoft Teams, si dispone di una Guida dettagliata automaticamente: [Elenco di controllo di accesso Guest](https://docs.microsoft.com/en-us/microsoftteams/guest-access-checklist)
  
Potrebbe essere necessario attendere fino a 24 ore per i team si applicano le modifiche di configurazione da eseguire.
