# <a name="whats-a-domain"></a>Che cos'è un dominio

Un dominio è la parte dell'indirizzo di posta elettronica che si trova dopo il simbolo @. Ad esempio, nell'indirizzo rob@contoso.com, il dominio è **contoso.com**. La maggior parte delle aziende preferisce usare il proprio nome di dominio nell'indirizzo di posta elettronica.

Di seguito sono riportati alcuni suggerimenti relativi ai domini:

- È possibile acquistare un nome di dominio da Microsoft. In questo modo, si dirà di disporre di un indirizzo di posta elettronica personalizzato.
- In caso contrario, è sufficiente aggiungere un dominio di cui si è già proprietari a Office 365.
