# <a name="onboard-a-windows-server-to-microsoft-defender-for-endpoint"></a>Eseguire l'onboard di un server Windows in Microsoft Defender per Endpoint

Per il sistema operativo Windows Server, Microsoft Defender for Endpoint offre funzionalità avanzate per rilevare e analizzare gli attacchi. Queste funzionalità vengono fornite senza problemi tramite la console di Microsoft Defender Security Center.

Defender for Endpoint supporta le versioni seguenti di Windows Server:

- Windows Server 2008 R2 SP1
- Windows Server 2012 R2
- Windows Server 2016
- Windows Server (SAC) versione 1803 e successive
- Windows Server 2019 e versioni successive
- Windows Server 2019 Core Edition

Per altre informazioni, vedere [Eseguire l'onboard dei server Windows nel servizio Microsoft Defender for Endpoint.](https://go.microsoft.com/fwlink/?linkid=2143627)
