# <a name="find-email-in-autoarchive"></a>Trovare la posta elettronica nell'archiviazione automatica

Il file di archivio è un tipo di file di dati di Outlook (con estensione pst). La prima volta che viene eseguita l'archiviazione automatica, il file di archivio viene creato automaticamente in: C:\Utenti\NomeUtente\Documenti\File di Outlook\archive.pst.

Per visualizzare l'archiviazione automatica:

1. In Outlook scegliere la **scheda File** > **Apri file** di dati di  >  **Outlook.**
2. Passare al percorso del file PST e aprirlo. Il file PST verrà aggiunto all'elenco delle cartelle.
3. È possibile cercare nella cartella i messaggi di posta elettronica passati.
