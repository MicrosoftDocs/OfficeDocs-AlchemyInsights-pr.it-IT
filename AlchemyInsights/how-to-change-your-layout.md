# <a name="how-to-change-your-layout"></a>Come modificare il layout

È possibile modificare le impostazioni in Outlook.com in modo che appaia e si comporta come desiderato. Selezionare le **Impostazioni** sulla barra degli strumenti per le opzioni più popolari.

Per visualizzare tutte le opzioni di layout, selezionare layout di posta **delle impostazioni**  >  **Mail**  >  [**Layout**](https://outlook.live.com/mail/options/mail/layout).
