# <a name="view-reports-on-sharepoint-activity-and-usage"></a>Visualizzare i report sull'attività e sull'utilizzo di SharePoint

Passare alla [pagina report sull'utilizzo nell'interfaccia di amministrazione di Microsoft 365](https://admin.microsoft.com/AdminPortal/Home). In **selezionare un report** scegliere il rapporto attività o utilizzo di SharePoint.

Per ulteriori informazioni, vedere [informazioni sui report di SharePoint](https://go.microsoft.com/fwlink/?linkid=875240).
