# <a name="site-is-not-available"></a>Sito non è disponibile

Se il sito di SharePoint o OneDrive non è disponibile a più utenti, potrebbe esserci un problema temporaneo del servizio. Controllare il[Dashboard sull'integrità del servizio](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) per verificare se è interessato il tenant.
