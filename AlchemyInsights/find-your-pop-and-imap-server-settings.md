# <a name="find-your-pop-or-imap-server-settings"></a>Trovare le impostazioni del server POP o IMAP

Se non si conoscono le impostazioni necessarie per aggiungere l'account POP o IMAP a Outlook, è possibile cercarli nelle [impostazioni di posta elettronica POP e IMAP per Outlook](https://support.office.com/article/8361e398-8af4-4e97-b147-6c6c4ac95353.aspx).
  
> [!NOTE]
> Lo sapevate che è possibile [abilitare e configurare POP3 su un server Exchange 2016](https://technet.microsoft.com/library/bb124934%28v=exchg.160%29.aspx)?
