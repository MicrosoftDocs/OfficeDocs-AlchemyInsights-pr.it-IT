# <a name="disabling-advanced-outlookcom-security"></a>Disabilitare la protezione avanzata Outlook.com

Per garantire la massima protezione per l'account, Outlook.com controlla se i collegamenti nei messaggi sono tentativi di phishing o sono probabile che scaricare virus o malware nel computer. Se si è sicuri di che voler questa protezione avanzata Outlook.com rimossa, scegliere **Sì** riportata di seguito.
