# <a name="profile-property-synchronization"></a>Sincronizzazione delle proprietà del profilo

Non è possibile eseguire il mapping di proprietà aggiuntive per l'importazione di Active Directory in modo che vengano sincronizzate dall'elenco di applicazioni profilo utente. L'importazione di Active Directory è limitata a un insieme preconfigurato di proprietà per garantire prestazioni coerenti del processo timer.
  
[Per ulteriori informazioni sulle proprietà del profilo sincronizzate](https://go.microsoft.com/fwlink/?linkid=875671)
