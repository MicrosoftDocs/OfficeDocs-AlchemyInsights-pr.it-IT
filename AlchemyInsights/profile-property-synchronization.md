# <a name="profile-property-synchronization"></a>Proprietà sincronizzazione dei profili

Non è possibile eseguire il mapping di proprietà aggiuntive per l'importazione di Active Directory per la sincronizzazione di Azure Active Directory per l'applicazione profili utente. Importazione Active Directory è limitato a un set di proprietà per garantire prestazioni coerenti del processo timer preconfigurato.
  
[Ulteriori informazioni sulle proprietà del profilo che di sincronizzazione](https://go.microsoft.com/fwlink/?linkid=875671)
