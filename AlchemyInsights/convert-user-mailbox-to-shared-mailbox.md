# <a name="convert-a-user-mailbox-to-a-room-or-equipment-mailbox"></a>Convertire una cassetta postale utente in cassetta postale della sala o in cassetta postale attrezzatura

È inoltre possibile convertire una cassetta postale da un tipo di cassetta postale a un altro, ma è necessario eseguire queste attività utilizzando PowerShell di Exchange Online. Per altre informazioni, vedere [Convertire una cassetta postale](https://go.microsoft.com/fwlink/p/?LinkId=832875).
