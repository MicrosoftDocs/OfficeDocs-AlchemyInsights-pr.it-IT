# <a name="convert-a-user-mailbox-to-a-room-or-equipment-mailbox"></a>Convertire una cassetta postale utente in cassetta postale della sala o in cassetta postale attrezzatura

È inoltre possibile convertire una cassetta postale dal tipo di una cassetta postale a un altro, ma è necessario eseguire queste attività tramite Exchange Online PowerShell. Estrarre [convertire una cassetta postale](https://go.microsoft.com/fwlink/p/?LinkId=832875) per ulteriori informazioni.
