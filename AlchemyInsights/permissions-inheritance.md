# <a name="how-permissions-inheritance-works-in-sharepoint"></a>Come funziona l'ereditarietà delle autorizzazioni in SharePoint

Per impostazione predefinita, le autorizzazioni in SharePoint vengono ereditate dall'alto nella gerarchia. Pertanto, un file eredita le autorizzazioni dalla cartella, che eredita le autorizzazioni dalla raccolta, che eredita le autorizzazioni dal sito, che eredita le autorizzazioni dalla raccolta siti.
  
Per informazioni sulla rimozione di autorizzazioni univoche e sul ripristino dell'ereditarietà, vedere [modificare e gestire le autorizzazioni per un elenco o una raccolta](https://go.microsoft.com/fwlink/?linkid=869946).
