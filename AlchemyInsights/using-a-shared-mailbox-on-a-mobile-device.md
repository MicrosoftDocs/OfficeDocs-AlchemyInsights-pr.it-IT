# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>Utilizzo di una cassetta postale condivisa in un dispositivo mobile

Non è possibile accedere a una cassetta postale condivisa dall'app Outlook installata sul dispositivo mobile. Questa funzionalità è utile? Informare il team di progettazione di Outlook. [Votare qui](https://go.microsoft.com/fwlink/?linked=862116)!
  
È possibile utilizzare una cassetta postale condivisa solo su un dispositivo mobile aprendo il browser, accedendo a Office 365 e quindi accedendo a Outlook sul Web. Per istruzioni, vedere [aggiungere una cassetta postale condivisa in Outlook sul Web](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207).
  
Se è necessario accedere alle cassette postali condivise utilizzando l'app Outlook su un dispositivo mobile, è consigliabile creare un gruppo di Office 365 anziché utilizzare una cassetta postale condivisa. Per ulteriori informazioni, vedere [confronto di gruppi](https://docs.microsoft.com/office365/admin/create-groups/compare-groups).
  
Per ulteriori informazioni, [vedere la Guida in-app per Outlook per iOS e Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
