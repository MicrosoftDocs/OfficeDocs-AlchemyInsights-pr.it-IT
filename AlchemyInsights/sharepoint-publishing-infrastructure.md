# <a name="sharepoint-publishing-infrastructure"></a>Infrastruttura di pubblicazione di SharePoint


SharePoint include una serie di funzionalità classiche che consentono di creare e pubblicare pagine Web ricche per la rete Intranet. Attivare l'infrastruttura di pubblicazione di SharePoint a livello di raccolta siti per attivarli.

Per ulteriori informazioni, visitare i collegamenti riportati di seguito.

[Attivare le caratteristiche di pubblicazione](https://support.office.com/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518)

[Funzionalità di pubblicazione](https://support.office.com/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553)

Note:

- I modelli di sito di pubblicazione classico includono le caratteristiche di pubblicazione per impostazione predefinita.

- Le caratteristiche di pubblicazione classiche non sono disponibili nei siti moderni. I siti moderni offrono un'alternativa più recente per impostazione predefinita.
