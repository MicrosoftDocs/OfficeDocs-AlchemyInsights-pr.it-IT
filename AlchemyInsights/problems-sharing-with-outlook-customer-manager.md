# <a name="problems-sharing-with-outlook-customer-manager"></a>Problemi di condivisione con gestione clienti di Outlook

Se si verificano problemi di condivisione con Outlook Customer Manager, è consigliabile provare a svuotare la cache del browser. Ad esempio in Internet Explorer è possibile:

1. Passare a **** > impostazioni**generali** > impostazioni nella**cache della** **cronologia** > esplorazioni e nei database.
2. Eliminare i file di cache per il sito Web "outlookapps.com".
3. Riavviare Outlook.

Per ulteriori informazioni sulla condivisione, vedere [Share Information in Outlook Customer Manager](https://support.office.com/article/4f26cc69-67da-4cd5-b344-02d1a4799310%20).
