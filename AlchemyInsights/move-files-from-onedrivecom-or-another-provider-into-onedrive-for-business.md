# <a name="move-files-from-onedrivecom-or-another-provider-into-onedrive-for-business"></a>Spostare i file da OneDrive.com o da un altro provider in OneDrive for Business

Se si ha accesso locale ai file, è possibile spostare i file in OneDrive for Business caricando i file in un browser Web o usando il client di sincronizzazione di OneDrive. Sono disponibili altri strumenti che consentono lo spostamento:

- Una panoramica completa dei modi in cui è possibile spostare i contenuto in OneDrive for Business. Per altre informazioni, vedere [Eseguire la migrazione dei contenuti a Microsoft 365](https://docs.microsoft.com/sharepointmigration/migrate-to-sharepoint-online).
- Per spostare i blocchi appunti di OneNote saranno necessari diversi passaggi. Per ulteriori informazioni, vedere [Spostare un blocco appunti di OneNote in OneDrive](https://support.office.com/article/move-a-onenote-notebook-to-onedrive-0af0a141-0bdf-49ab-9e50-45dbcca44082).
    
- Abilitare File su richiesta. È possibile usare i file su richiesta sia per Windows che per Mac. Per altre informazioni, vedere [Risparmiare spazio sul disco con i File di OneDrive su richiesta per Windows 10](https://support.office.com/article/Save-disk-space-with-OneDrive-Files-On-Demand-for-Windows-10-0e6860d3-d9f3-4971-b321-7092438fb38e) e [Risparmiare spazio su disco con i File OneDrive su richiesta per Mac](https://support.office.com/article/Save-disk-space-with-OneDrive-Files-On-Demand-for-Mac-529f6d53-e572-4922-a585-e7a318c135f0).
- È possibile eseguire il backup delle cartelle importanti nel PC Windows (ovvero le cartelle Desktop, Documenti e Immagini) con il Backup delle cartelle del PC di Microsoft OneDrive, così da renderle protette e disponibili in altri dispositivi. Per altre informazioni, vedere [Eseguire il backup delle cartelle Documenti, Immagini e Desktop con OneDrive](https://support.office.com/article/back-up-your-documents-pictures-and-desktop-folders-with-onedrive-d61a7930-a6fb-4b95-b28a-6552e77c3057) e [Come OneDrive protegge i dati dell'utente nel cloud](https://support.office.com/article/how-onedrive-safeguards-your-data-in-the-cloud-23c6ea94-3608-48d7-8bf0-80e142edd1e1). 
- Quando si spostano i file condivisi in OneDrive for Business, vengono riconosciuti come nuovi file, quindi è consigliabile condividerli di nuovo. Per altre informazioni, vedere [Condividere file in OneDrive for Business](https://support.office.com/article/Move-files-from-OneDrive-to-OneDrive-for-Business-7fb28cad-7e25-451f-8b4b-2d1a71e5c0e9#sharefiles).
