# <a name="microsoft-stream-licensing-overview"></a>Panoramica della gestione delle licenze di Microsoft Stream

Microsoft Stream è un servizio cloud concesso in licenza su base di sottoscrizione per utente. È disponibile come parte delle sottoscrizioni di Office 365 idonee. Per informazioni su ogni tipo di sottoscrizione, vedere Office 365 pricing.
