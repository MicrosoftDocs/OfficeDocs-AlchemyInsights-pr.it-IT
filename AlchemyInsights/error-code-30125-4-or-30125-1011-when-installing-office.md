# <a name="error-code-30125-4-or-30125-1011-when-installing-office"></a>Codice di errore 4 30125 o 30125 1011 durante l'installazione di Office

È possibile che questo errore se il software antivirus, firewall, le impostazioni del proxy o connessione impedire l'installazione di Office. Ecco alcuni aspetti che è possibile provare a risolvere il problema:
  
- Riavviare il computer e reinstallare Office
    
- Installazione di Office tramite il [programma di installazione non in linea di Office](https://support.office.com/article/f0a85fe7-118f-41cb-a791-d59cef96ad1c?wt.mc_id=Alchemy_ClientDIA)
    
- Disattivare temporaneamente le impostazioni del proxy
    
- Disattivare temporaneamente il software antivirus
    
- Disattivare temporaneamente il firewall
    
Per ulteriori informazioni, vedere [il codice di errore 4 30125 o 30125 1011 durante l'installazione di Office](https://support.office.com/article/7bfabec6-76be-4cde-880e-819a9c569612?wt.mc_id=Alchemy_ClientDIA)
