# <a name="limit-sharing-in-microsoft-365"></a>Limitare la condivisione in Microsoft 365

Anche se non è possibile disabilitare completamente la condivisione interna o rimuovere il pulsante Condividi dai siti, è possibile limitare la condivisione in Microsoft 365 in modo da soddisfare le esigenze dell'organizzazione. Per ulteriori informazioni, vedere [limitare la condivisione in Microsoft 365](https://docs.microsoft.com/Office365/Enterprise/microsoft-365-limit-sharing).
