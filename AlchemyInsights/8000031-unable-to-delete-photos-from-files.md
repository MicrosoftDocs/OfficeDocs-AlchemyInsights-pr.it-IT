# <a name="unable-to-delete-photos-from-files"></a>Non è possibile eliminare le foto da file

Per eliminare il file o le foto:
  
1. Passare alla cartella **posta in arrivo** o **Posta inviata** ed eliminare il messaggio di posta elettronica che sono associati. 
    
2. Rimuovere gli elementi dagli **elementi eliminati**. 
    
Per ulteriori informazioni, informazioni su [come eliminare i file o le foto di Outlook.com](https://support.office.com/article/bae0531f-040f-4c42-90b9-786ca718c16d.aspx).
