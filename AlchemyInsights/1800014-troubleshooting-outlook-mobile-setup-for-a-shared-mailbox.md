# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>Risoluzione dei problemi di configurazione per dispositivi mobili di Outlook per una cassetta postale condivisa

È possibile accedere alla cassetta postale condivisa da Outlook app installata sul dispositivo mobile. Specificare se questa funzionalità. Informare il team di progettazione di Outlook. [Di seguito voto](https://go.microsoft.com/fwlink/?linked=862116)!
  
È possibile utilizzare solo una cassetta postale condivisa su un dispositivo mobile per aprire il browser, effettuare l'accesso a Office 365 e quindi passare Outlook sul web. Per ulteriori informazioni, vedere [aggiunta di una cassetta postale condivisa in Outlook sul web](https://support.office.com/en-us/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207). Da Outlook sul web sarà in grado di accedere alla cassetta postale condivisa.
  
Ulteriori informazioni: [Guida all'interno dell'applicazione per Outlook per iOS e Android](https://support.office.com/en-us/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
