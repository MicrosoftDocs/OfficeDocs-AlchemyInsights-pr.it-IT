# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Ottenere un account disattivata per errore in Outlook sul Web?

Se viene visualizzato l'errore in `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` cui l' **account è stato disabilitato** quando si tenta di aprire Outlook sul Web (in precedenza noto come Outlook Web App), l'amministratore dell'organizzazione potrebbe aver disabilitato l'accesso a Outlook sul Web. L'amministratore può correggere questo errore attenendosi alla procedura descritta nell'argomento [abilitare o disabilitare Outlook Web App per una cassetta postale](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx).
