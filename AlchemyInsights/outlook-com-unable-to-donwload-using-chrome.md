# <a name="cant-download-files-in-chrome"></a>Non è possibile scaricare i file in Chrome

**Problema** Si tratta di un problema che impedisce ad alcuni clienti di scaricare file quando si utilizza il browser Google Chrome. 

**Soluzione alternativa** Utilizzare [Microsoft Edge](https://www.microsoft.com/windows/microsoft-edge) o un altro Web browser.
Per ulteriori informazioni, vedere [download di file e allegati di Outlook.com](https://support.office.com/article/8d7c1ea7-4e5f-44ce-bb6e-c5fcc92ba9ab?wt.mc_id=Office_Outlook_com_Alchemy).
