# <a name="change-your-account-picture"></a>Modificare l'immagine dell'account

1. Visualizzare [le informazioni sull'account Microsoft Your](https://go.microsoft.com/fwlink/p/?linkid=860841).
    
2. Selezionare **Cambia immagine**. 
    
3. Selezionare **nuova immagine**. 
    
4. Scegliere l'immagine che si desidera utilizzare, quindi scegliere **Apri**. 
    
5. Trascinare o ridimensionare la casella di immagine e selezionare **Salva**. 
    
Consentire fino a 24 ore per le foto dei profili o nome rendere effettive le modifiche.
