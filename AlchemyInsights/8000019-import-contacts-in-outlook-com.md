# <a name="import-contacts"></a>Importazione di contatti

Prima di importare contatti per Outlook.com, sarà necessario esportarli dal proprio altri provider di posta elettronica come file CSV. Seguire le istruzioni riportate di seguito.
  
1. Nella parte inferiore della pagina, selezionare **persone**. 
    
2. Nella barra dei menu, scegliere **Gestione** \> **Importa contatti**. 
    
3. Selezionare **Gmail** (anche se il file con estensione csv potrebbe non provengono da Gmail) perché Gmail viene utilizzato il tipo di file. csv. 
    
4. Scegliere **Sfoglia**, passare alla cartella in cui è stato salvato contatti esportati, quindi scegliere **Apri**. 
    
Per ulteriori informazioni, vedere [importare contatti a un account Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873136).
