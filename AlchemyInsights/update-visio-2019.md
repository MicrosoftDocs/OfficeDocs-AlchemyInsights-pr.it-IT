# <a name="upgrade-to-visio-2019"></a>Eseguire l'aggiornamento a Visio 2019

Se è stata installata l'app desktop di Visio come parte dell'abbonamento a Visio online, è già presente la versione più recente. Per confrontare i piani di Visio e le opzioni non relative alla sottoscrizione, vedere confrontare le opzioni di Visio.

Se si dispone di una versione non di sottoscrizione di Visio, ad esempio Visio 2016 e si desidera eseguire l'aggiornamento a Visio 2019, è necessario acquistare Visio 2019 o acquistare un abbonamento a Visio online. Se si dispone già di Visio 2019 e si è pronti per l'installazione, vedere [Install Visio](https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1798). 

Se si esegue l'installazione affiancata di Visio con Office nello stesso computer, vedere [scenari supportati per l'installazione di versioni diverse di Office, Project e Visio nello stesso computer](https://docs.microsoft.com/deployoffice/install-different-office-visio-and-project-versions-on-the-same-computer).

**Nota**: la versione principale di Visio 2019 si trova a 16,0.
