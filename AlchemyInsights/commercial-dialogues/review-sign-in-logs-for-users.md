# <a name="review-sign-in-logs-for-users"></a>Esaminare i log di accesso per gli utenti

Ecco come eseguire questa operazione:

1. Passare a **Interfaccia di amministrazione** Azure Active  >  **[Directory](https://go.microsoft.com/fwlink/p/?linkid=2067268)**.
1. Selezionare **Azure Active Directory**  >  **Sign-ins**.
1. Se non si dispone già di una sottoscrizione di Azure AD P1 o P2, verrà richiesto di avviare una versione di valutazione premium gratuita. Se si desidera, fare clic **su Avvia** una versione di valutazione Premium gratuita e quindi nella casella Azure AD Premium P2 selezionare Attiva versione di **valutazione**  >  **gratuita.** La versione di valutazione gratuita è di 30 giorni.
