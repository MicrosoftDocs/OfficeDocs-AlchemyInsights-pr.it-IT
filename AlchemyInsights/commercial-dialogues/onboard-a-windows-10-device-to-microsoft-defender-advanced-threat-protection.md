# <a name="onboard-a-windows-10-device-to-microsoft-defender-advanced-threat-protection-mdatp"></a>Onboard di un dispositivo Windows 10 a Microsoft Defender Advanced Threat Protection (MDATP)

MDATP supporta gli strumenti e i metodi di distribuzione seguenti per Windows 10:

- Criteri di gruppo
- Microsoft Endpoint Configuration Manager
- Gestione dei dispositivi mobili, incluso Microsoft Intune
- Script locali

Per altre informazioni, vedi Strumenti e metodi [di onboarding per i dispositivi Windows 10.](https://go.microsoft.com/fwlink/?linkid=2143460)
