# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Versione supportata dello strumento di migrazione di SharePoint



Attualmente lo strumento di migrazione di SharePoint supporta solo le migrazioni da SharePoint 2013. Le versioni 2010 e 2016 non sono supportate in questo momento.
  
Per ulteriori informazioni, vedere [Introduzione allo strumento di migrazione di SharePoint](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
