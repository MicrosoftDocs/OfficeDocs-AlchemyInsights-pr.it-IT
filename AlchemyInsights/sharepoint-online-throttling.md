# <a name="sharepoint-online-throttling"></a>Limitazione di SharePoint Online

SharePoint Online utilizza la limitazione per mantenere le prestazioni ottimali e l'affidabilità del servizio SharePoint Online. La limitazione limita il numero di azioni utente o di chiamate simultanee (tramite script o codice) per impedire un utilizzo eccessivo delle risorse. Se si ottiene la limitazione, il 99% del tempo è a causa del codice personalizzato.
  
Per ulteriori informazioni sulla limitazione, vedere [impedire la limitazione o il blocco in SharePoint Online](https://go.microsoft.com/fwlink/?linkid=2022019)
