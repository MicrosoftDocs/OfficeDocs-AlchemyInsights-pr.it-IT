# <a name="how-to-type-special-characters-on-a-mac"></a>Come digitare caratteri speciali su un Mac

Per digitare caratteri speciali nel Mac, aprire il menu **caratteri speciali** con la combinazione di tasti di scelta rapida **comando**⌘  +  **Control**  +  **Space**.

Un collegamento rapido per @ è anche **ALT**  +  **64**.
