# <a name="solutions-for-unsupported-operating-system"></a>Soluzioni per "Sistema operativo non supportato"



Quando si tenta di installare Office 365, è possibile che un messaggio di errore di **sistema operativo non supportato** se il dispositivo non soddisfa i [requisiti di sistema](https://products.office.com/office-system-requirements).
  
Office 365, è necessario:
  
- PC: 10 Windows, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 o Windows Server 2008 R2
    
    OPPURE
    
- Mac OS X 10.10
    
Se è stato visualizzato il messaggio durante il tentativo di installare Office in un dispositivo mobile o un Chromebook, vedere [come installare ed eseguire Microsoft Office in un Chromebook](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
