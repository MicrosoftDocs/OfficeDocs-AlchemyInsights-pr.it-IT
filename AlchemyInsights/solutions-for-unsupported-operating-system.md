# <a name="solutions-for-unsupported-operating-system"></a>Soluzioni per "Sistema operativo non supportato"

Quando si tenta di installare Office 365/2016, è possibile che il messaggio di errore se il dispositivo non soddisfa i [requisiti di sistema](https://products.office.com/office-system-requirements).
  
 **"Sistema operativo non supportato"**
  
Office 365/2016, è necessario:
  
PC: 10 Windows, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 o Windows Server 2008 R2
  
OPPURE
  
Mac OS X 10.10
  
Se è stato visualizzato il messaggio durante il tentativo di installare Office in un dispositivo mobile o un Chromebook, vedere [Set up Office apps e posta elettronica di un dispositivo mobile](https://support.office.com/article/7dabb6cb-0046-40b6-81fe-767e0b1f014f?wt.mc_id=Alchemy_ClientDIA.aspx).
