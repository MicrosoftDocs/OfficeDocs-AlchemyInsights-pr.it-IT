# <a name="list-view-threshold"></a>Soglia visualizzazione elenco

La soglia di visualizzazione è pensata per garantire le prestazioni del servizio. La soglia non può essere cambiata in SharePoint Online. Per evitare di raggiungere questa soglia, vedere [Gestire elenchi ampi e librerie in SharePoint](https://support.office.com/article/manage-large-lists-and-libraries-in-sharepoint-b8588dae-9387-48c2-9248-c24122f07c59).

È anche possibile che venga visualizzato questo errore quando si prova a condividere o interrompere l'ereditarietà di una cartella. Vedere [Errore di condivisione/interruzione dell'ereditarietà](https://docs.microsoft.com/SharePoint/troubleshoot/lists-and-libraries/error-share-break-inheritance).
