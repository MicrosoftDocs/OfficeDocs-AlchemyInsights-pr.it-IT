# <a name="publishing-infrastructure-with-a-communication-site"></a>Infrastruttura di pubblicazione a un sito di comunicazione


Attivare la caratteristica infrastruttura di pubblicazione di SharePoint Server è **non è supportato** con siti di comunicazione. 
  
Per ulteriori informazioni, vedere [passaggio dalla pubblicazione dei siti ai siti di comunicazione](https://docs.microsoft.com/sharepoint/publishing-sites-classic-to-modern-experience).
