# <a name="workflows-in-sharepoint"></a>Flussi di lavoro in SharePoint

I flussi di lavoro di SharePoint sono applicazioni preprogrammate che semplificano una vasta gamma di processi aziendali. Per altre informazioni, vedere i collegamenti riportati di seguito.

## <a name="getting-started"></a>Attività iniziali

Se si sta appena iniziando con i flussi di lavoro di SharePoint, vedere [Intro to SharePoint workflows](https://support.office.com/article/introduction-to-sharepoint-workflow-07982276-54e8-4e17-8699-5056eff4d9e3).

## <a name="troubleshoot-and-debug-a-sharepoint-workflow"></a>Risoluzione dei problemi e debug di un flusso di lavoro di SharePoint

Vedere [risolvere gli errori di convalida del flusso di lavoro di SharePoint](https://docs.microsoft.com/sharepoint/dev/general-development/troubleshooting-sharepoint-server-workflow-validation-errors-in-visio) e il [debug dei flussi di lavoro](https://docs.microsoft.com/sharepoint/dev/general-development/debugging-sharepoint-server-workflows)

## <a name="power-automate"></a>Automatizzare la potenza

Considerare l'utilizzo di [Power automatizzate](https://docs.microsoft.com/power-automate/modern-approvals) anziché delle soluzioni di flusso di lavoro di SharePoint classiche.
