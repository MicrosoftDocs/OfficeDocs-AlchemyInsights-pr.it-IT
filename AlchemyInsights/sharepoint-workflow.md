# <a name="workflows-in-sharepoint"></a>Flussi di lavoro in SharePoint

I flussi di lavoro di SharePoint sono mini-applicazioni preprogrammate che semplificano e automatizzano una vasta gamma di processi aziendali. I flussi di lavoro possono variare dalla raccolta di firme, commenti o approvazioni per un piano o documento, per tenere traccia dello stato corrente di una procedura di routine.

**Guida introduttiva:** Se si sta appena iniziando con i flussi di lavoro di SharePoint, vedere [Introduzione ai flussi di lavoro di SharePoint per altre informazioni](https://support.office.com/article/introduction-to-sharepoint-workflow-07982276-54e8-4e17-8699-5056eff4d9e3).

**Microsoft Flow**: valutare la possibilità di utilizzare Microsoft Flow anziché le soluzioni di flusso di lavoro di SharePoint classiche. Per altre informazioni, vedere [creare un flusso per un elenco o una raccolta in SharePoint Online o OneDrive](https://support.office.com/article/create-a-flow-for-a-list-or-library-in-sharepoint-online-or-onedrive-for-business-a9c3e03b-0654-46af-a254-20252e580d01)

**Risoluzione dei problemi relativi al flusso di lavoro e debug**:

Il debug dei flussi di lavoro è simile a quello delle versioni precedenti. Per altre informazioni, vedere [debug dei flussi di lavoro di SharePoint](https://docs.microsoft.com/sharepoint/dev/general-development/debugging-sharepoint-server-workflows). Per ulteriori informazioni sulla risoluzione dei problemi, vedere [risolvere i problemi relativi alla convalida del flusso di lavoro di SharePoint Server](https://docs.microsoft.com/sharepoint/dev/general-development/troubleshooting-sharepoint-server-workflow-validation-errors-in-visio).
