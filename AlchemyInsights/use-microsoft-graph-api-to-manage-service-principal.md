# <a name="use-microsoft-graph-api-to-manage-service-principal"></a>Utilizzare l'API di Microsoft Graph per gestire l'entità servizio

Con l' [API Principal del servizio](https://docs.microsoft.com/graph/api/resources/serviceprincipal), è possibile gestire a livello di programmazione le istanze delle applicazioni e controllare le operazioni che un'applicazione può eseguire all'interno del tenant.

Per ulteriori informazioni sulla relazione tra le applicazioni e le entità di servizio, vedere [Applications and Service Principal Objects in Azure Active Directory](https://docs.microsoft.com/azure/active-directory/develop/app-objects-and-service-principals).
