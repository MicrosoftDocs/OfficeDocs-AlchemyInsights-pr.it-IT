# <a name="install-project-professional"></a>Installare Project Professional

Vedere [Install Project](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281) per informazioni su come installare la versione di sottoscrizione di Project online e sulla versione non di sottoscrizione di Project 2019, 2016 o 2013. 

Per Project 2010 o 2007, seguire la procedura descritta in [Office 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d) o [Office 2007](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657). 

Il progetto viene venduto come applicazione autonoma, quindi se non è ancora disponibile, [acquistare o provare Project](https://products.office.com/project).
