# <a name="microsoft-search-setup-guide"></a>Guida alla configurazione di Microsoft Search

Usare la [guida alla configurazione di Microsoft Search](https://go.microsoft.com/fwlink/?linkid=2153798) per configurare Microsoft Search. Se si desidera eseguire la distribuzione pilota a un gruppo di utenti o stentarla a tutti gli utenti dell'organizzazione. Si assegnano gli amministratori della ricerca e gli editor di ricerca e quindi si personalizza l'esperienza di ricerca per gli utenti con risposte e opzioni aggiuntive, come l'aggiunta dell'estensione Bing a Chrome o l'impostazione di Bing come motore di ricerca predefinito.
