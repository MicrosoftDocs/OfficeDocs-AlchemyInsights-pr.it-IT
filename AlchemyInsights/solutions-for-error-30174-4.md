# <a name="error-code-30174-4-when-installing-office"></a>Codice di errore 4 30174 durante l'installazione di Office

È possibile che questo errore se il software antivirus, firewall, le impostazioni del proxy o connessione impedire l'installazione di Office. Ecco alcuni aspetti che è possibile provare a risolvere il problema.
  
- Utilizzare una connessione cablata
    
- Disattivare temporaneamente le impostazioni del proxy
    
- Disattivare temporaneamente il software antivirus
