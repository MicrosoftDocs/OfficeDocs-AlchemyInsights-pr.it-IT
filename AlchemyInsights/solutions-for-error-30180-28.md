# <a name="solutions-for-error-30180-28"></a>Soluzioni per errori 30180 28

È possibile che [errore 28 30180](https://support.office.com/article/47ae453b-677c-412f-9a21-6766555ff4de?wt.mc_id=Alchemy_ClientDIA) durante un aggiornamento di Office se si dispone di una connessione internet insufficiente. Ecco alcuni aspetti che è possibile provare a risolvere il problema. 
  
- Riavviare il computer e quindi [aggiornare Office](https://support.office.com/article/2ab296f3-7f03-43a2-8e50-46de917611c5?wt.mc_id=Alchemy_ClientDIA).
    
- Eseguire la procedura precedente, ma utilizzare una connessione cablata durante l'aggiornamento di Office.
