# <a name="how-to-change-your-theme"></a>Come modificare il tema

Per modificare il tema Outlook.com o Outlook sul web, passare a **Impostazioni**![impostazioni](media/f4b2e798-fff1-4a14-931f-5677a4543b58.png) e scegliere un tema in **tema**. Per visualizzare tutti i temi disponibili, selezionare **Visualizza tutto** in **tema**.
