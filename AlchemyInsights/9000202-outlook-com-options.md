# <a name="how-to-find-options-in-outlookcom-settings"></a>Come trovare le opzioni nelle impostazioni di Outlook.com

Nella parte superiore della pagina, selezionare l'icona ingranaggio per aprire le impostazioni. Nella parte inferiore del riquadro delle impostazioni, selezionare **Visualizza tutte le impostazioni di Outlook**.

In questo modo vengono visualizzate le impostazioni generali, la posta elettronica, il calendario e le opzioni di premio (se applicabile).
