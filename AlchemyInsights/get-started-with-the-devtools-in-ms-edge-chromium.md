# <a name="get-started-with-the-developer-tools-in-microsoft-edge-chromium"></a>Iniziare a utilizzare gli strumenti di sviluppo in Microsoft Edge (Chromium)

Microsoft Edge ha adottato il progetto Chromium open-source, che mira a rendere il Web meno frammentato e le piattaforme web più compatibili. Questa modifica consentirà all'utente di utilizzare Microsoft Edge per creare e testare i siti Web che funzionano su altri browser basati su cromo.

Microsoft Edge (Chromium) fornisce anche [strumenti di sviluppo di qualità ottimale](https://go.microsoft.com/fwlink/?linkid=2134941) che dovrebbero essere familiari e compatibili con gli strumenti già utilizzati.

Per ulteriori informazioni, vedere [What ' s New in the Microsoft Edge (Chromium) devtools](https://go.microsoft.com/fwlink/?linkid=2135020).
