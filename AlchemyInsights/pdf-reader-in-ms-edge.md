# <a name="pdf-reader-in-microsoft-edge"></a>Lettore di file PDF in Microsoft Edge

La popolarità dei file PDF evidenzia la necessità di un lettore PDF affidabile, sicuro e potente che può essere adottato dalle aziende dell'organizzazione. Il lettore di file PDF ricco di funzionalità integrato in Microsoft Edge (versioni 77 e successive) è solo un'app di questo tipo. Funziona in Windows e macOS e può essere utilizzato per aprire e annotare i file PDF locali, online o incorporati nelle pagine Web.

Per ulteriori informazioni, vedere [PDF Reader in Microsoft Edge](https://go.microsoft.com/fwlink/?linkid=2140005).
