# <a name="onedrive-is-stuck-on-processing-changes"></a>OneDrive è bloccato nelle modifiche all'elaborazione

Se OneDrive Visualizza "modifiche all'elaborazione" per un lungo periodo di tempo, potrebbe essere perché si dispone di un file online aperto, un numero elevato di file bloccati in una coda o una sincronizzazione di file di grandi dimensioni. Read  [OneDrive è bloccato su "modifiche all'elaborazione"](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd) per i passaggi consigliati e altre informazioni.

Se si ha bisogno di ulteriore assistenza, è possibile  [inviare messaggi di posta elettronica OneDrive](https://go.microsoft.com/fwlink/p/?LinkId=528676).
