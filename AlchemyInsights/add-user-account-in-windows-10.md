# <a name="add-a-user-account-in-windows-10"></a>Aggiungere un account utente in Windows 10

Ecco come aggiungere un account in Windows 10:

1. Andare a **Start** > **Settings** > **accounts** > **famiglia & altre persone** (o **altre persone**, se si utilizza Windows 10 Enterprise) > **aggiungere qualcun altro a questo PC**.
2. Seguire le istruzioni per aggiungere un account Microsoft o un account locale.

[Ulteriori informazioni sulla configurazione degli account](https://support.microsoft.com/help/17197/)
