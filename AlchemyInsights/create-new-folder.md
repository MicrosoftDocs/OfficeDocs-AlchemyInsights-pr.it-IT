# <a name="create-a-new-folder"></a>Creare una nuova cartella

1. In **cartelle** nel riquadro di spostamento selezionare **nuova cartella**.

2. Digitare un nome per la cartella e premere INVIO.

Per ulteriori informazioni, vedere [creazione di una cartella in Outlook.com](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a).
