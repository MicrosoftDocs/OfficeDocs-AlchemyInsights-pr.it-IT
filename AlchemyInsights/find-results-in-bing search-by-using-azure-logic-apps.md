# <a name="find-results-in-bing-search-by-using-azure-logic-apps"></a>Trovare i risultati in Bing Search usando app per la logica di Azure

Usando Bing Search dall'interno di un'app per la logica, puoi automatizzare l'elaborazione dei risultati della ricerca e renderli disponibili per altre azioni. Ad esempio, puoi trovare notizie e quindi fare in modo che Twitter le inseri come tweet nel feed.

Per altre informazioni, vedere [Trovare i risultati in Ricerca Bing tramite App per la logica di Azure.](https://go.microsoft.com/fwlink/?linkid=2151928)
