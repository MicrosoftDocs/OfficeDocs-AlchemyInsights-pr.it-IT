# <a name="550-57705-access-denied-tenant-has-exceeded-threshold"></a>550 5.7.705 accesso negato, tenant ha superato soglia

Questo messaggio viene generato quando troppo posta posta indesiderata o blocco è stato inviato dall'organizzazione Office 365. Per rimuovere il blocco, è necessario comprendere e spiegano la causa di un agente di supporto, nonché risolvere il problema.
