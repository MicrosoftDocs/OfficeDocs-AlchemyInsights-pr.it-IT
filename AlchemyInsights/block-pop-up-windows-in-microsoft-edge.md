# <a name="block-pop-up-windows-in-microsoft-edge"></a>Bloccare le finestre popup in Microsoft Edge

1. Vai a **... (Impostazioni e altro)**  >  **Impostazioni**  >  **Autorizzazioni sito**.
2. Selezionare **Popup e reindirizza**.
3. Attiva **Blocco**.
