# <a name="learn-about-external-calendar-sharing-options"></a>Informazioni sulle opzioni di condivisione dei calendari esterni

[! Nota] se è necessario pianificare riunioni con persone esterne all'organizzazione, Microsoft 365 consente di rendere disponibili i calendari per gli "utenti esterni", coloro che devono visualizzare i tempi di disponibilità ma non gli account utente per l'ambiente Microsoft 365. Per ulteriori informazioni, vedere [condividere calendari con utenti esterni](https://docs.microsoft.com/office365/admin/manage/share-calendars-with-external-users).
