# <a name="how-to-reactivate-a-subscription"></a><span data-ttu-id="20707-102">Come riattivare un abbonamento</span><span class="sxs-lookup"><span data-stu-id="20707-102">How to reactivate a subscription</span></span>

<span data-ttu-id="20707-103">Se l'abbonamento è scaduto, è stato annullato o è stato disabilitato da Microsoft, è possibile riattivarlo nell'interfaccia di amministrazione di Microsoft 365.</span><span class="sxs-lookup"><span data-stu-id="20707-103">If your subscription has expired, been cancelled, or disabled by Microsoft, you can reactivate it in the Microsoft 365 admin center.</span></span>
  
1. <span data-ttu-id="20707-104">Nell'interfaccia di amministrazione passare alla pagina **Fatturazione** \> [I tuoi prodotti](https://go.microsoft.com/fwlink/p/?linkid=842054).</span><span class="sxs-lookup"><span data-stu-id="20707-104">In the admin center, go to the **Billing** \> [Your products](https://go.microsoft.com/fwlink/p/?linkid=842054) page.</span></span>
2. <span data-ttu-id="20707-105">Nella scheda **Prodotti**, trovare l'abbonamento da riattivare.</span><span class="sxs-lookup"><span data-stu-id="20707-105">On the **Products** tab, find the subscription that you want to reactivate.</span></span> <span data-ttu-id="20707-106">Selezionare **Altre azioni** (tre puntini), quindi selezionare **Riattiva l’abbonamento**.</span><span class="sxs-lookup"><span data-stu-id="20707-106">Select **More actions** (three dots), then select **Reactivate this subscription**.</span></span>
    <span data-ttu-id="20707-107">Se l'opzione **Riattiva l’abbonamento** non è disponibile, contattare il supporto.</span><span class="sxs-lookup"><span data-stu-id="20707-107">If you don't see **Reactivate this subscription** as an available action, call support.</span></span>
3. <span data-ttu-id="20707-108">Nel riquadro **Riattiva l’abbonamento**, selezionare una modalità di pagamento nell'elenco a discesa oppure selezionare **Aggiungi modalità di pagamento**.</span><span class="sxs-lookup"><span data-stu-id="20707-108">In the **Reactivate this subscription** pane, select a payment method from the drop-down list, or select **Add payment method**.</span></span>
4. <span data-ttu-id="20707-109">Selezionare **Salva**.</span><span class="sxs-lookup"><span data-stu-id="20707-109">Select **Save**.</span></span>
