# <a name="upload-files-to-onedrive-or-sharepoint"></a>Caricare file in OneDrive o SharePoint

<p>È possibile caricare file in una raccolta documenti in SharePoint Online semplicemente trascinandole dal computer e rilasciandoli nella raccolta documenti. Con OneDrive, è possibile sincronizzare i file tra il computer e il cloud, in modo da poter accedere ai file da qualsiasi luogo. È inoltre possibile utilizzare i file sincronizzati direttamente in Esplora file e accedere ai file anche quando si è&rsquo;in modalità non in linea.
