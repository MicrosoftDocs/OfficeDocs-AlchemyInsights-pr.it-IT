1. <span data-ttu-id="d69d0-104">[Reimpostare la password](https://support.office.com/article/7a5d073b-7fae-4aa5-8f96-9ecd041aba9c) immediatamente.</span><span class="sxs-lookup"><span data-stu-id="d69d0-104">[Reset the user's password](https://support.office.com/article/7a5d073b-7fae-4aa5-8f96-9ecd041aba9c) immediately.</span></span> <span data-ttu-id="d69d0-105">Non comunicare la nuova password tramite posta elettronica all'utente finale.</span><span class="sxs-lookup"><span data-stu-id="d69d0-105">Do not communicate the new password through email to the end user.</span></span>

2. <span data-ttu-id="d69d0-106">Rimuovere qualsiasi [indirizzo di inoltro](https://support.office.com/article/ab5eb117-0f22-4fa7-a662-3a6bdb0add74) sospetto a livello di cassetta postale.</span><span class="sxs-lookup"><span data-stu-id="d69d0-106">Remove any suspicious [forwarding addresses](https://support.office.com/article/ab5eb117-0f22-4fa7-a662-3a6bdb0add74) set at the mailbox level.</span></span>

3. <span data-ttu-id="d69d0-107">Rimuovere tutte le [regole di posta in arrivo](https://support.office.com/article/1433E3A0-7FB0-4999-B536-50E05CB67FED) sospette all'interno della cassetta postale.</span><span class="sxs-lookup"><span data-stu-id="d69d0-107">Remove any suspicious [inbox rules](https://support.office.com/article/1433E3A0-7FB0-4999-B536-50E05CB67FED) set within the mailbox.</span></span>

4. <span data-ttu-id="d69d0-108">Se l'utente è bloccato e non può inviare la posta elettronica, [andare alla pagina Utenti con restrizioni per sbloccare l'account](https://protection.office.com/?hash=/restrictedusers).</span><span class="sxs-lookup"><span data-stu-id="d69d0-108">If the user is blocked from sending email, [go to the Restricted Users to unblock the account](https://protection.office.com/?hash=/restrictedusers).</span></span> <span data-ttu-id="d69d0-109">Fatto questo, dovrebbe essere possibile riprendere l'invio dei messaggi entro un'ora.</span><span class="sxs-lookup"><span data-stu-id="d69d0-109">Once done, the user should be able to resume sending messages within 1 hour.</span></span>

5. <span data-ttu-id="d69d0-110">Rimuovere l'account dell’utente da tutti i [gruppi di ruoli amministrativi](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504) finché non si è certi che l'account non è più compromesso.</span><span class="sxs-lookup"><span data-stu-id="d69d0-110">Remove the user account from any [administrative role groups](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504) until you are confident that the account is no longer compromised.</span></span>
