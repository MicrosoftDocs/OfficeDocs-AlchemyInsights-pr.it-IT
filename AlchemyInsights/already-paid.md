# <a name="already-paid"></a>Già pagato?

Se è già stata inviata pagamento, è possibile Inviaci:
  
1. Passare alla pagina [degli abbonamenti](https://go.microsoft.com/fwlink/p/?linkid=842054) di oppure scegliere **fatturazione** \> **sottoscrizioni**.
    
2. Scegliere una sottoscrizione e nella sezione **pagamento** **Visualizza distinte**.
    
3. Nella pagina **dettagli degli effetti** scegliere **già pagato?** Verrà chiesto di immettere le informazioni relative a pagamento che è stato inviato. 
    
4. Quando sono stato immesso tutte le informazioni necessarie, scegliere **Invia**.
