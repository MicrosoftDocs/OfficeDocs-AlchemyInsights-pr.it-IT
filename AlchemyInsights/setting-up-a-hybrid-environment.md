# <a name="setting-up-a-hybrid-environment"></a>Configurazione di un ambiente ibrido

Per consentire agli utenti di lavorare insieme senza problemi, indipendentemente dal fatto che i relativi dati vengano archiviati nei server locali o in Office 365, potrebbe essere necessario configurare un ambiente ibrido. Per ottenere una panoramica delle opzioni, vedere integrazione di [Office 365 con ambienti locali](https://support.office.com/article/263faf8d-aa21-428b-aed3-2021837a4b65.aspx).
