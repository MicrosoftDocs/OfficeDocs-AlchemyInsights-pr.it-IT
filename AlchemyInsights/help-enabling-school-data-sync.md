# <a name="help-enabling-school-data-sync"></a>Assistenza per l'implementazione di School Data Sync

Serve aiuto per abilitare School Data Sync? Si potrebbe essere idonei per ricevere assistenza nell'ambito del programma Microsoft 365 FastTrack.

Visitare il [FastTrack Center](https://www.microsoft.com/fasttrack) per inviare una richiesta di assistenza per abilitare School Data Sync per l'organizzazione.
