# <a name="learn-about-site-compatibilityaffecting-changes-coming-to-microsoft-edge"></a>Informazioni sulla compatibilità dei siti, che influiscono sulle modifiche apportate a Microsoft Edge

Il Web è in continua evoluzione per migliorare l'esperienza, la sicurezza e la privacy degli utenti. Alcune modifiche sono abbastanza significative per influire sulla funzionalità delle pagine esistenti. Il team Microsoft Edge aggiorna le pagine man mano che si evolve il pensiero, le sequenze temporali si solidificano e [vengono annunciate modifiche](https://go.microsoft.com/fwlink/?linkid=2135534).
