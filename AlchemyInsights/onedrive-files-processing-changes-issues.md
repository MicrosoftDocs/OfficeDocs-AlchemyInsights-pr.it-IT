# <a name="onedrive-troubleshoot-processing-changes"></a>OneDrive: risoluzione dei problemi relativi alle modifiche all'elaborazione

Se si visualizzano le "modifiche all'elaborazione" per un periodo di tempo prolungato, è possibile che si disponga di un file online aperto, di un numero elevato di file bloccati in una coda o di una sincronizzazione di file di grandi dimensioni, vedere [OneDrive Stuck on Processing](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd).

Leggere questo articolo per risolvere i problemi relativi alle modifiche all'elaborazione. Questo messaggio verrà visualizzato anche se si dispone di un file che cambia per tutto il tempo (come Outlook. File PST o un database). Informazioni su come [spostare i file di dati di Outlook](https://support.office.com/article/how-to-remove-an-outlook-pst-data-file-from-onedrive-b6b9e522-59bd-40f7-949f-168d0aa9b38e) che si sta utilizzando regolarmente fuori OneDrive.
