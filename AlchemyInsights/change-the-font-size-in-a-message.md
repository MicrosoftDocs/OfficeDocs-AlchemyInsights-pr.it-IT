# <a name="change-the-font-size-in-a-message"></a>Modificare la dimensione del tipo di carattere in un messaggio

1. Se il messaggio è già stato digitato, selezionare il testo.
2. Selezionare l'icona  **formato** al di sotto del messaggio (è la prima icona).
3. Selezionare l'icona della  **dimensione del carattere**  sopra il messaggio (è l'icona settima).
4. Selezionare una dimensione.
