# <a name="how-to-customize-permissions-for-a-sharepoint-list-or-library"></a>Come personalizzare le autorizzazioni per un elenco o una raccolta di SharePoint

In qualità di amministratore o proprietario di una raccolta, un elenco o un sondaggio, è possibile modificare le autorizzazioni per consentire a un gruppo di utenti di accedere ai dati necessari per limitare gli altri.

[Personalizzare le autorizzazioni per un elenco o una raccolta di SharePoint](https://support.office.com/article/customize-permissions-for-a-sharepoint-list-or-library-02d770f3-59eb-4910-a608-5f84cc297782)
