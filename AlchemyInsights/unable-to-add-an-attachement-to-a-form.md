# <a name="unable-to-add-an-attachment-to-an-externally-facing-form"></a>Impossibile aggiungere un allegato a un modulo con orientamento esterno

Se si sta tentando di aggiungere un allegato a un sondaggio di Microsoft Forms esterno, questa funzionalità non è ancora disponibile. Per monitorare lo stato della richiesta di funzionalità, passare a [Questa voce di UserVoice](https://go.microsoft.com/fwlink/?linkid=2133069)e aggiungere il proprio voto.
