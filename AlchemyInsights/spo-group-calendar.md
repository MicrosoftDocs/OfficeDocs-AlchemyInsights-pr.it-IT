# <a name="spo-group-calendar"></a>Calendario di gruppo di SharePoint Online

La Web part Calendario di gruppo consente di inserire un calendario di gruppo di Microsoft 365 direttamente nella pagina per renderlo facilmente visibile ai lettori.
- Per altre informazioni, vedere [Usare la Web part Calendario di gruppo](https://support.microsoft.com/en-us/office/use-the-group-calendar-web-part-eaf3c04d-5699-48cb-8b5e-3caa887d51ce?ui=en-us&rs=en-us&ad=us).
