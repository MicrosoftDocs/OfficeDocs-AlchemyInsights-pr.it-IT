# <a name="error-onedrive-cannot-connect-to-windows"></a>Errore: OneDrive non è in grado di connettersi a Windows

Per risolvere il messaggio di errore, cercare il file cldflt. sys per verificare se è in esecuzione e, se non lo è, attivarlo. 
  
Per ulteriori informazioni su come abilitare il file cldflt. sys, vedere [OneDrive non è in grado di connettersi a Windows "errore durante l'accesso ai file in OneDrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
