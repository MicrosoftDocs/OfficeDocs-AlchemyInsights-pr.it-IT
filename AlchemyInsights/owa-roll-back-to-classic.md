# <a name="the-new-outlook-on-the-web"></a>Nuovo Outlook sul Web

Non è più possibile tornare all'esperienza classica di Outlook sul Web. Se si desidera inviare commenti e suggerimenti sulla nuova esperienza, visitare il [UserVoice Forum](https://outlook.uservoice.com/forums/313228--outlook-on-the-web-office-365). È possibile votare le idee o i problemi che si desidera visualizzare e rispondere direttamente ai commenti e suggerimenti principali.

[Ottenere assistenza per la nuova Outlook sul Web](https://support.office.com/article/017014cd-2ad0-41ab-8473-6bd8c349d4f8)
