# <a name="deploy-and-update-microsoft-365-apps-with-configuration-manager-advisor"></a>Distribuire e aggiornare le app di Microsoft 365 con Configuration Manager Advisor

Per le organizzazioni che utilizzano Configuration Manager, è possibile utilizzare le [app deploy e Update Microsoft 365 con Configuration Manager Advisor](https://admin.microsoft.com/adminportal/home#/oppinstall) per generare uno script che configuri automaticamente la distribuzione delle app Microsoft 365 utilizzando le procedure consigliate da ingegneri di FastTrack.

Utilizzare questa guida per creare i gruppi di distribuzione, personalizzare le app e le funzionalità di Office, configurare installazioni dinamiche o Lean, quindi eseguire lo script per la creazione di applicazioni, regole di distribuzione automatiche e raccolte di dispositivi necessari per la distribuzione.
