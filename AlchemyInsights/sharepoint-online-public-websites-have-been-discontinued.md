# <a name="sharepoint-online-public-websites-have-been-discontinued"></a>I siti Web pubblici di SharePoint Online sono stati interrotti

<p><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">I siti Web pubblici di SharePoint Online sono stati interrotti.&nbsp;</span></p> <p><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Se è stato creato un sito Web pubblico di SharePoint Online, è stato possibile continuare a utilizzarlo al 31 marzo 2018. Se si desidera continuare a utilizzare un sito Web pubblico, è necessario crearne uno nuovo con un nuovo provider. Il contenuto del sito Web pubblico di SharePoint Online è stato eliminato.&nbsp;</span></p> <p><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Per informazioni sulla sospensione del sito Web pubblico, vedere <a href="https://go.microsoft.com/fwlink/?linkid=866980">informazioni dettagliate sul processo di sospensione delle funzionalità</a>.</span></p>
