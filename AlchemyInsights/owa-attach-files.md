# <a name="how-to-attach-files-to-an-email"></a>Come allegare i file a un messaggio di posta elettronica

1. Nella parte inferiore di un messaggio o di un evento del calendario, selezionare l'icona graffetta.
1. Selezionare **Sfoglia questo computer** per collegare un file nel computer o sfogliare le **posizioni del cloud** per allegare un file online.

Per ulteriori informazioni, vedere [allegare file in Outlook sul Web](https://support.office.com/article/48b8dca1-7a76-43ce-97d1-e1cf73893f55).
