# <a name="delete-a-folder"></a>Eliminare una cartella

1. Nel riquadro sinistro fare clic sulla cartella o sottocartella che si desidera eliminare e scegliere **Elimina**. 
    
2. Scegliere **OK** per confermare. 
    
Per ulteriori informazioni, vedere [eliminazione di una cartella in Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873134).
