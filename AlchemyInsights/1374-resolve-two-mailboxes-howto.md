# <a name="a-user-has-two-mailboxes"></a>Un utente dispone di due cassette postali

Ambienti ibridi che utilizzano Azure Active Directory connettersi (connessione AAD) o DirSync accidentalmente potrebbero provocare un utente di cassette postali due: uno dell'organizzazione locale e una nel cloud.
  
Per risolvere questo problema, vedere [Definitivamente informazioni della cassetta postale precedente non crittografato](https://blogs.technet.microsoft.com/exchange/2018/01/17/permanently-clear-previous-mailbox-info/).
