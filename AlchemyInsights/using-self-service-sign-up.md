# <a name="using-self-service-sign-up-in-your-organization"></a>Uso dell'iscrizione self-service nell'organizzazione

Con l'iscrizione self-service gli utenti possono iscriversi per usare servizi compresi nell'abbonamento dell'organizzazione.

- Per controllare se gli utenti possono eseguire l'iscrizione per gli abbonamenti in modalit√† self-service, usare il cmdlet [Set-MsolCompanySettings](https://docs.microsoft.com/powershell/module/msonline/set-msolcompanysettings?view=azureadps-1.0) di PowerShell con il parametro **AllowAdHocSubscriptions**. Per altre informazioni, vedere [Come si controllano le impostazioni self-service](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/self-service-purchase-faq?view=o365-worldwide)?

- Per vedere tutti i programmi disponibili, consultare [Programmi self-service disponibili](https://docs.microsoft.com/microsoft-365/admin/misc/self-service-sign-up?view=o365-worldwide#available-self-service-programs).

Per le informazioni complete, vedere [Uso dell'iscrizione self-service nell'organizzazione](https://docs.microsoft.com/microsoft-365/admin/misc/self-service-sign-up?view=o365-worldwide).
