# <a name="issues-seeing-calendar-freebusy-information"></a>Problemi relativi alle informazioni sulla disponibilità del calendario

Se non è possibile visualizzare le informazioni sulla disponibilità del calendario in altri calendari di Outlook o se altre persone non sono in grado di visualizzare le informazioni sulla disponibilità nel calendario, utilizzare l' [Assistente di supporto e ripristino di Microsoft per microsoft 365](https://diagnostics.office.com/).
