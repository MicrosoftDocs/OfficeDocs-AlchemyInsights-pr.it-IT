# <a name="view-reports-on-onedrive-activity-and-usage"></a>Visualizzare i report sull'attività e sull'utilizzo di OneDrive

Passare alla [pagina report sull'utilizzo nell'interfaccia di amministrazione di Microsoft 365](https://admin.microsoft.com/AdminPortal/Home). In **selezionare un report** scegliere l'attività o il report sull'utilizzo di OneDrive.

Per ulteriori informazioni, vedere [informazioni sui report di OneDrive](https://go.microsoft.com/fwlink/?linkid=875239).
