# <a name="sync-sharepoint-files-with-the-new-onedrive-sync-client"></a>Sincronizzare i file di SharePoint con il nuovo client di sincronizzazione di OneDrive

<p>
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Il comando Apri con Esplora risorse apre un'istanza locale di Windows Explorer che mostra la struttura delle cartelle sul server che ospita il sito di SharePoint. Detto questo, è consigliabile <a href="https://support.office.com/it-IT/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88">sincronizzare i file di SharePoint con il nuovo client di sincronizzazione di OneDrive</a>, che fornisce <a href="https://support.office.com/it-IT/article/learn-about-onedrive-files-on-demand-0e6860d3-d9f3-4971-b321-7092438fb38e">File su richiesta</a> perché permette l'accesso locale ai file e offre le prestazioni migliori</span></p> <p><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Se si è scelto di usare la visualizzazione Esplora file invece di usare il nuovo client di sincronizzazione, assicurarsi di seguire i passaggi e le procedure consigliate descritti negli articoli seguenti.</span></p> <ul> <li>
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">
  <a href="https://support.office.com/it-IT/article/How-to-use-the-Open-with-Explorer-command-to-troubleshoot-issues-in-SharePoint-Online-87155331-0c92-4224-a4c1-da5c21c4ade4">Come usare il comando "Apri con Esplora risorse" per risolvere i problemi in SharePoint Online</a></span></li> <li>
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">
  <a href="https://support.office.com/it-IT/article/copy-or-move-library-files-by-using-open-with-explorer-aaee7bfb-e2a1-42ee-8fc0-bcc0754f04d2">Copiare o spostare file delle raccolte tramite Apri con Esplora risorse</a></span></li> </ul> <p><strong style="mso-bidi-font-weight: normal;"><u><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Nota:</span></u></strong><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;"> il pulsante Apri con Esplora risorse non è presente nella nuova interfaccia della raccolta. Fare clic sull'elenco a discesa <strong>Visualizzazione</strong> in alto a destra (il nome di questo elenco cambia a seconda della visualizzazione corrente) e quindi fare clic cu <strong>Visualizza</strong> in Esplora risorse</span>.</p> <p><strong style="mso-bidi-font-weight: normal;"><u><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Nota:</span></u></strong><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;"> Apri con Esplora risorse di SharePoint usa i <strong>controlli ActiveX</strong>, quindi è supportato solo in Internet Explorer 10 o 11. Non funziona in Windows con Microsoft Edge, Google Chrome, Mozilla Firefox o sulla piattaforma Mac. Per questo motivo, l'opzione &ldquo;Visualizzazione Esplora risorse&rdquo; può essere inattiva.</span></p> <ul> <li>
