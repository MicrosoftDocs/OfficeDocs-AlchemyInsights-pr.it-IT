# <a name="learn-about-mapping-between-google-chrome-policies-and-microsoft-edge-policies"></a>Informazioni sul mapping tra i criteri di Google Chrome e i criteri Microsoft Edge

I criteri di Google Chrome sono mappati ai criteri rilevanti di Microsoft Edge supportati nella versione 80 del browser. Per ulteriori informazioni, vedere [mapping dei criteri di Google Chrome a Microsoft Edge](https://go.microsoft.com/fwlink/?linkid=2141933).
