# <a name="open-and-use-a-shared-mailbox-in-outlook-on-the-web"></a>Aprire e usare cassette postali condivise in Outlook sul web per le aziende

Le cassette postali condivise a cui si ha accesso non vengono visualizzate automaticamente in Outlook sul web (in precedenza noto come Outlook Web App).

È possibile visualizzare sia la cassetta postale condivisa che la cassetta postale principale o aprire la cassetta postale condivisa in una finestra separata del browser. Per ricevere notifiche tramite messaggi per la cassetta postale condivisa, è necessario aprire la cassetta postale condivisa in una finestra separata del browser. Per altre informazioni, vedere [Aprire e usare cassette postali condivise in Outlook Web App](https://support.office.com/article/Add-a-shared-mailbox-to-Outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207).
