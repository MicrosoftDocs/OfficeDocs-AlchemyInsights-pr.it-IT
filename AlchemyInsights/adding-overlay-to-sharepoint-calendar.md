# <a name="adding-an-overlay-to-a-sharepoint-calendar"></a>Aggiunta di una sovrapposizione a un calendario di SharePoint

I calendari di SharePoint sono tipi di elenco classici che a volte sono confusi con i calendari di gruppo di Microsoft 365.
 
Se si utilizza un calendario di SharePoint nel sito, è possibile sovrapporlo al proprio calendario di Exchange per creare una visualizzazione combinata del calendario. In alternativa, è possibile sovrapporre un altro calendario di SharePoint purché entrambi i calendari di SharePoint si trovino nella stessa raccolta siti.
 
Per aggiungere una sovrapposizione a un calendario di SharePoint classico, seguire la procedura illustrata in sovrapposizione di un calendario di [SharePoint con un calendario di Exchange o SharePoint](https://support.office.com/article/Overlay-a-SharePoint-calendar-with-a-calendar-from-Exchange-or-SharePoint-4CAEBE59-3994-4A94-9322-B31ABB8A5E9A).
