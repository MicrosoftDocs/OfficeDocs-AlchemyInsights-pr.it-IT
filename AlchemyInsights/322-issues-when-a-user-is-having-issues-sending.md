# <a name="a-user-is-having-issues-sending-email-messages"></a>Un utente si verificano problemi l'invio di messaggi di posta elettronica

Sembra che è necessario un problema di flusso di posta elettronica. È disponibile la diagnostica che consente di identificare più comuni problemi di flusso di posta elettronica. Fare clic su **esecuzione della diagnostica** riportata di seguito.
