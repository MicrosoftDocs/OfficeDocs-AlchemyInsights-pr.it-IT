# <a name="copy-files-in-sharepoint-and-onedrive"></a>Copiare i file in SharePoint e OneDrive

È possibile copiare i file dal Web browser in posizioni diverse all'interno di SharePoint e OneDrive.

Per ulteriori informazioni, vedere [Move or copy](https://support.microsoft.com/office/00e2f483-4df3-46be-a861-1f5f0c1a87bc)

> [!NOTE]
> **Copia in** non è disponibile se si utilizza l'esperienza classica di SharePoint Online.
