# <a name="sync-a-users-profile-picture"></a>Sincronizzare profili immagine di un utente

Immagine sincronizzazione dei profili in SharePoint Online richiede un'azione dell'utente di avviare la sincronizzazione delle loro foto. Non esiste alcuna sincronizzazione automatica delle foto per SharePoint Online. SharePoint Online Sincronizza solo le foto di Exchange Online, non Azure Active Directory.
  
Se un utente non ha mai eseguito un'azione che richiesta una sincronizzazione oppure hanno aggiornato le immagini del profilo e il nuovo uno non viene visualizzata in SharePoint Online, l'utente potrebbe essere necessario chiedere conferma che la sincronizzazione.
