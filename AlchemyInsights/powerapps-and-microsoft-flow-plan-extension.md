# <a name="powerapps-and-microsoft-flow-plan-extension"></a>PowerApps e Microsoft Flow Plan Extension

Se si intende richiedere un'estensione di piano per **Microsoft Flow** o **PowerApps** e il tenant ha già utenti attivi di una delle seguenti funzionalità, non è necessario presentare una richiesta di supporto.

- Creazione e pubblicazione di connettori personalizzati in PowerApps e Microsoft Flow
- Azioni personalizzate HTTP integrate nel flusso all'esterno di SharePoint e OneDrive
- Integrazione dei dati locali tramite il gateway di dati locale

I clienti con utenti attivi di queste funzionalità avranno un'estensione automatica fino al 31 gennaio 2020 o la scadenza del periodo di sottoscrizione di Microsoft 365 esistente.
